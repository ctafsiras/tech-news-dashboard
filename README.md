# Tech News Dashboard

A responsive React + TypeScript dashboard that fetches technology headlines from NewsAPI.org. Features:

- Responsive Bootstrap card grid
- Category filter (AI, Programming, Gadgets)
- Live search through headlines
- Bookmarking to localStorage
- Dark mode toggle (Bootstrap data theme)

## Quick Start

1. Install dependencies:

   ```bash
   npm install
   ```

2. Configure environment:

   - Copy `.env.example` to `.env.local`
   - Set your NewsAPI key in `VITE_NEWS_API_KEY`

3. Run the app:

   ```bash
   npm run dev
   ```

4. Open http://localhost:5173

Notes:

- In development, requests go through a Vite proxy that injects the API key as an `X-Api-Key` header to avoid CORS issues.
- In production, the app calls NewsAPI directly and sends the key via header.

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
